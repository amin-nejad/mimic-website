MIMIC-III DATASET

MIMIC is an openly available dataset developed by the MIT Lab for Computational Physiology, comprising deidentified health data associated with ~60,000 intensive care unit admissions. It includes demographics, vital signs, laboratory tests, medications, and textual notes. Read more here: Physionet

TRANSFORMERS

State-of-the-art (SOTA) solutions attempt to solve NLG tasks using neural-network based architectures. Since Vaswani et al (2017), this has mostly been done using the Transformer architecture. Its features such as multihead attention have been shown to be be ideal for NLG.

SO WHAT?

In this work, we train a number of Transformer models on discharge summaries from the MIMIC dataset for NLG purposes. On this website, we allow users to input information regarding a patient, choose which model they would like, and generate a discharge summary relevant to that patient.
